# 404

### (╯°Д°)╯︵ ┻━┻

页面**丢进异次元**辣 ！！！

<script>
    var i = 15;  //这里是倒计时的秒数
    var intervalid;
    intervalid = setInterval("cutdown()", 1000);
    function cutdown() {
        if (i == 0) {
          window.location.href = "/"; //倒计时完成后跳转的地址
            clearInterval(intervalid);
        }
        document.getElementById("mes").innerHTML = i;
        i--;
    }
    window.onload = cutdown;
</script>

页面将在<span id="mes"></span>秒后自动跳转。
